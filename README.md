# Portofolio Kenny Karli Architect

Website ini dibuat menggunakan [Next.js](https://nextjs.org) untuk menampilkan portofolio proyek-proyek Kenny Karli Architect.

## Menjalankan di Lokal

Jalankan server development dengan perintah berikut:

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser untuk melihat hasilnya.

## Fitur Utama

- Halaman beranda dengan deskripsi profil kantor arsitek
- Tentang kantor arsitek dan galeri proyek
- Responsif untuk desktop dan mobile

## Teknologi

- Next.js
- React
- Tailwind CSS untuk styling
